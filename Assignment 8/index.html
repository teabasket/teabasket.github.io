About this app

For my app I have decided to use the 5 key interface with (UP, DOWN, RIGHT, LEFT) arrows and a select key this will enable the user to use a lesser number of keys than there are buttons on the app. As an experimentation, I made the LEFT key 'a', RIGHT key 'd', UP key 'w', and DOWN key 's' and the enter key 'f'. This way the user will only have to use one hand. I then made another program with 5 key interface this time making the direction the arrow keys and and the spacebar the select key. To compare these two interfaces I picked 2 equations and timed myself while I entered it into the calculator. I did 5 different Trials.

  • Instructions for using the app:
    DOWN: 's' key
    UP: 'w' key
    LEFT: 'a' key
    RIGHT: 'd' key
    SELECT: 'f' key
